# echo-cli
CLICK para el framework hecho de Golang

## Instalacion
    go get -u github.com/Obsinqsob01/echo-cli

## Uso

    En la terminal corres:
	    echo-cli -<command>=<value>
	    
		Ejemplo:
			echo-cli -new=blog
